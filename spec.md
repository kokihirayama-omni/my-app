# Todoリストアプリ 仕様書

## 概要
HTMLとJavaScriptのみで構築するシンプルなTodoリストアプリケーション

## 機能要件

### 1. Todoアイテムの追加
- 入力フォームからTodoアイテムを追加できる
- 入力後、Enterキーまたは追加ボタンで登録
- 空の入力は登録できない
- 登録後、入力フィールドはクリアされる

### 2. Todoアイテムの表示
- 追加されたTodoアイテムをリスト形式で表示
- 各アイテムには以下を含む：
  - チェックボックス（完了/未完了の切り替え）
  - Todoのテキスト
  - 削除ボタン

### 3. Todoアイテムの完了/未完了切り替え
- チェックボックスをクリックで完了/未完了を切り替え
- 完了したアイテムは視覚的に区別（取り消し線など）

### 4. Todoアイテムの削除
- 削除ボタンをクリックでアイテムを削除

### 5. データの永続化
- localStorageを使用してデータを保存
- ページをリロードしてもデータが保持される

## 技術要件

### ファイル構成
- `index.html` - HTMLマークアップとスタイル
- `script.js` - JavaScript処理

### 使用技術
- HTML5
- CSS3（インラインまたはstyleタグ）
- Vanilla JavaScript（フレームワーク不使用）
- localStorage API

## UI/UX要件

### デザイン
- シンプルで使いやすいインターフェース
- レスポンシブデザイン（モバイルでも使用可能）
- 見やすい配色とフォント

### ユーザー体験
- 直感的な操作
- 素早いレスポンス
- わかりやすいフィードバック

## データ構造

### Todoアイテム
```javascript
{
  id: Number,           // 一意のID
  text: String,         // Todoのテキスト
  completed: Boolean    // 完了状態
}
```

### localStorage保存形式
- キー: `todos`
- 値: Todoアイテムの配列をJSON文字列化したもの

## 実装の優先順位
1. 基本的なHTML構造とスタイル
2. Todoの追加機能
3. Todoの表示機能
4. 完了/未完了の切り替え機能
5. 削除機能
6. localStorageでのデータ永続化
